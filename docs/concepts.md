# Core Concepts

## Package
A primitive installable ingredient (e.g. `chromium`, `pnpm`, `uv`).

## Capability
A composed outcome‑oriented unit (e.g. `web-automation`, `data-science`).

## Alias
A friendly shortcut for a capability or package (e.g. `scrape`).

## Recipe
A YAML file describing what to install or run.

## Registry
A GitHub‑hosted index + recipes.
