# CLI Reference

## Search
```bash
agentkit search <query>
```

## Inspect
```bash
agentkit inspect <name>
```

## Add
```bash
agentkit add <name>
agentkit add <name> --dry-run
```

## List
```bash
agentkit list packages
agentkit list capabilities
```

## Flags
- `--json` — structured output
- `--dry-run` — plan only
- `--yes` — non‑interactive
- `--cwd` — run in a specific directory
