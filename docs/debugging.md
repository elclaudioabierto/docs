# Debugging

When something fails, you want deterministic evidence.

## Environment report
```bash
agentkit env
```

For full details:
```bash
agentkit env --json
```

## Doctor
```bash
agentkit doctor
```

## Logs
By default, step output is saved to:
```
.agentkit/agentkit.log
```
