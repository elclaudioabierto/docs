# Roadmap

## Now (live)

- **Execution step runners**: run/install/copy/check/open with clean logs
- **Deterministic recipes**: inspect before running
- **Registry‑backed capabilities**: GitHub‑native sources
- **Registry cards + search**: richer browsing UX for humans and agents

## Next up (1–3 weeks)

- **Template rendering**: scaffold full project layouts with variables
- **Explain plans**: human‑readable summary + risks before execution

## Later (after core stability)

- **Private registry auth** (not shipped yet)
- **Registry validation tooling**: lint recipes before publishing
- **Deterministic caching**: avoid repeat work across agents

If you want a feature moved up, open an issue with a concrete use‑case.
